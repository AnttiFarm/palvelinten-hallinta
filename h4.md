## x) Karvinen 2023: Configuration Management of Distributed Systems over Unreliable and Hostile Networks

- 4.12.1 Size and Complexity of Some DSLs:
  - Salt DSL käyttää 510 tila funktiota, näillä funktioilla on yli 20 000 linjaa dokumentaatiotekstiä. Puppet DSL käyttää 113.
- 4.12.2 Use of DSL Functions in Case Configuration:
  - Puppetissa Mozillan manifestien mukaan 12 komentoa kattaa 87% kaikista käytetyistä komennoista. Yleisimpiä funktioita mitä käytetään ovat, exec, package, file ja service. Vaikka on todella monia komentoja ja funktioita, pienellä määrällä niitä pärjää pitkälle.
- 4.12.3.1 Dependencies Between Main Functions:
  - File ja exec funktiot ovat tärkeimmät ja niiden päälle voi rakentaa lisää esim. package user ja group.

## a) Räpylä

Asensin postgresql/postgres tietokannan hallintajärjestelmän. Aloitin lataamalla sen komennoilla `sudo apt-get update` ja `sudo apt-get install postgresql`. Testasin että se tuli ladattua komennolla `sudo systemctl status postgresql`.

Koska se oli ladattuna ja päällä aloitin sen toiminnan testaamisen. Loin uuden tietokannan `sudo -u postgres createdb antti` ja uuden käyttäjän tämän hallintaa varten `sudo -u postgres createuser antti`. Sitten käynnistin hallintaohjelman `psql` komennolla.

Testasin toimintaa ensin `\?` ja `\echo moi`. Näistä tuli selville että postgres oli asennettu ja toimintakunnossa.
